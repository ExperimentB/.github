# Experiment B
### The Minecraft Backrooms

**Experiment B** is a Minecraft server aiming to recreate the full backrooms lore in-game!  
In an expansive world, you fight off entities while making sure you yourself don't die in the brutal conditions in the backrooms.  
Travel through many different levels, survive to the best of your ability, and found communities to make your permanent mark on these liminal halls.
